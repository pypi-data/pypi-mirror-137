Resting state and noise acquisition¶

Implemented through extensions systems in the RestingNoiseAcquisition module.

This experiment perform the same task implemented in 1: Resting with open eyes, Resting with close eye, Eye blinking, Eyeball movement up/down, Eyeball movement left/right, Head movement left/right and Jaw clenching. The time acquisition time for each task can be configured independently. The subject can explicitly. The subject can explicitly decide when to continue with the next task with a keyboard event.

Configuration options¶

All tasks have the same configuration option, the acquisition time, from 0 (disabled task) to 5 minutes, 1 minute by default.

1
